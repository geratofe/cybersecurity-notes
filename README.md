# Linux Commands for Cybersecurity

This repository contains my personal notes on Linux commands while learning cybersecurity and penetration testing.

The goal is to build a reference of useful commands, examples, and practical use cases.

## What is inside this repository

The repository contains markdown files explaining different Linux commands and how they can be used.

Examples include:

- ls
- cd
- pwd
- cat
- echo
- touch & mkdir
- cp
- mv
- rm
- shred
- chmod
- grep
- find

Each file includes:
- basic usage
- useful options
- examples
- sometimes pentesting related usage

## Why I made this

While learning cybersecurity I noticed that many commands repeat across labs and CTF challenges. Writing them down helps me remember them and gives me a quick reference when practicing.
